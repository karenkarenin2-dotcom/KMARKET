# KMARKET

**Аналитика рынка World of Warcraft. Отвечает на один вопрос: брать жетон сейчас или ждать.**

KMARKET by KareninTeam · регион EU (+ US как опережающий индикатор)

---

## Что это

Blizzard отдаёт через свой API только **текущую** цену жетона — никакой истории.
KMARKET копит эту историю сам, круглосуточно, и превращает её в ответ:

- **Вердикт** — брать / ждать / не сейчас, с объяснением, почему именно так.
- **Перцентиль цены** — «сейчас 12-й перцентиль за 90 дней» честнее, чем «дёшево».
- **Сезонность** — тепловая карта час × день недели по времени EU-серверов:
  когда жетон исторически дешевле всего.
- **Бэктест** — сколько золота правило реально сэкономило бы за прошлый год
  против покупки наугад. Совет, подтверждённый цифрой, а не ощущением.
- **Алерты в Telegram** — когда цена вошла в интересную зону.

## Как устроено

Три независимых слоя:

1. **Сборщик** — GitHub Actions, каждые 10 минут, ноль зависимостей и ноль затрат.
   Пишет историю прямо в этот репозиторий. Работает, даже когда компьютер выключен.
2. **Аналитика** — чистые функции над накопленной историей.
3. **Дашборд** — локальный веб-интерфейс, запускается ярлыком с рабочего стола.

История лежит в `data/history/<регион>/<ГГГГ-ММ>.csv` — обычные CSV,
которые можно открыть чем угодно.

## Запуск

```bash
# 1. Секреты (ключи — на https://develop.battle.net/access/clients)
cp .env.example .env   # и заполнить

# 2. Разовый сбор вручную — зависимостей не требует
python -m kmarket.collect

# 3. Дашборд
pip install -r requirements.txt
python -m kmarket.web
```

## Правила игры

KMARKET читает официальный Blizzard Game Data API, который Blizzard сама
предоставляет разработчикам. Игрового клиента проект не касается и не
автоматизирует — никаких аддонов, макросов и вмешательства в игру.
